void RestResponse_use_buffer ( RestResponse *  self,
char *  buffer,
size_t  buffer_size 
)

This function can be used to preconfigure a static response buffer to capture the response.

This can be used to improve memory efficiency especially when downloading files by reusing the same buffer mutliple times. This buffer will be set on the body member of the RestResponse object and will NOT be freed by the destructor.

Parameters:
selfthe RestResponse to modify.
bufferthe buffer to use for the response.
buffer_sizethe size of the buffer.
